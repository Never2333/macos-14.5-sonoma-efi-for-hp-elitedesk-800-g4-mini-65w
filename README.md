# macOS 14.5+ Sonoma EFI For HP Elitedesk 800 G4 mini 65W
OpenCore EFI configuration for hackintosh using HP Elitedesk 800 G4 Mini 65W, installing macOS Sonoma 14.5.

## 机器配置 Configuration

| 硬件  Items       | 型号  Model               |
| ----------- | ------------------- |
| CPU         | Intel Core i7-8700 |
| RAM         |  DDR4 32GB 3200 MHz (16GB x2) |
| iGPU | Intel UHD 630 |
| SSD | QUANXING (铨兴) N301 |
| Sound Card  | Conexant CX20632      |
| Ethernet    | Intel I219-LM           |
| WLAN and Bluetooth | Intel AX210 |


## EFI 文件组成和来源  EFI files Composition and Sources

本EFI配置文件部分由 [RapidEFI](https://github.com/JeoJay127/RapidEFI-Tool) 生成，部分 ACPI 文件来自 [10h30 的 EFI文件](https://github.com/10h30/macos-sonoma-efi-opencore-hp-elitedesk-800-g4-g5-mini)。

This EFI configuration file was partially generated by [RapidEFI](https://github.com/JeoJay127/RapidEFI-Tool), and part of the ACPI file is from [EFI file from 10h30](https://github.com/10h30/macos-sonoma-efi-opencore-hp-elitedesk-800-g4-g5-mini).

Kexts 包含 AirportItlwm.kext、AppleALC.kext、BlueToolFixup.kext、IntelBluetoothFirmware.kext、IntelBTPatcher.kext、IntelMausiEthernet.kext、Lilu.kext、RestrictEvents.kext、SMCProcessor.kext、SMCSuperIO.kext、USBInjectAll.kext、VirtualSMC.kext、WhateverGreen.kext。

Kexts contain: AirportItlwm.kext、AppleALC.kext、BlueToolFixup.kext、IntelBluetoothFirmware.kext、IntelBTPatcher.kext、IntelMausiEthernet.kext、Lilu.kext、RestrictEvents.kext、SMCProcessor.kext、SMCSuperIO.kext、USBInjectAll.kext、VirtualSMC.kext、WhateverGreen.kext.

SMBIOS 选用 Macmini8,1，需要自行填写三码以确保某些苹果服务的正常使用。

You need to configure UUID, MLB, and SerialNumber yourself to ensure the use of some Apple services.

## 使用教程  Installation

使用前，请给原来的 EFI 文件备个份。

Please BACKUP the original EFI files before using these EFI files.


一般地，使用 [OpenCore Configurator](https://mackie100projects.altervista.org/download-opencore-configurator/) 挂载 EFI 分区，然后将本库的 EFI 文件覆盖进去即可。

Mount EFI partition using [OpenCore Configurator](https://mackie100projects.altervista.org/download-opencore-configurator/), then replace the EFI folder with this repository EFI files. Normally it works.

更多 OpenCore 教程请访问 https://dortania.github.io/OpenCore-Install-Guide/

For more tips, please refer to https://dortania.github.io/OpenCore-Install-Guide/

## 可用功能  Available Functions

- [x] 内置扬声器 Onboard sound card
- [x] 板载网卡 Onboard network card
- [x] Wifi
- [x] 睡眠/唤醒 Sleep / Wake up*
- [x] USB 2.0 / USB 3.0 及 USB-C 端口
- [x] 蓝牙 Bluetooth**
- [X] 小版本 OTA 升级 OTA upgrade (within Sonoma)
- [X] HDMI 音频 HDMI audio
- [ ] DRM
- [ ] AirDrop / SideCar / Handoff

*睡眠/唤醒 Sleep / Wake up：存在一些问题，需要重新拔插 HDMI 线缆才可成功唤醒，且重新拔插后，在 2K 显示器上使用 1080p 的 HiDPI 会失效，重启可解决问题。There are some issues, you need to replug the HDMI cable to wake up successfully, and after replugging, HiDPI with 1080p on a 2K display will not work, reboot may solve the problem.

**蓝牙 Bluetooth：目前仅测试可发现其他设备，未测试其功能是否正常。Currently only testing for other devices to be found, not testing for specific functionality.
